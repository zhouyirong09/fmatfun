## Flexible matlab functions
## 介绍
fmatfun 是一个灵活易用的Matlab函数集，通过将经常使用的功能进行封装形成，涉及到作图、数据、文本、文件等，编写目的是提高使用Matlab和生成科技报告写作材料的效率。

目前，可使用的函数如下。
* [matfig](matfig) 用于作图的函数
* [demo](demo) 如何使用fmatfun范例

## 安装
* 下载 fmatfun
* 在Matlab中运行 [install.m](install.m)
